Exportar ejercicios

Generá un ZIP con todos los ejercicios del sistema (predeterminados y propios). Incluye datos (JSON) e imágenes en carpeta imagenes/. Sirve para backup o para restaurar en otra instalación.

Importar ejercicios

Subí un archivo ZIP exportado desde este sistema. Las imágenes y los datos se restauran en sus carpetas y en la base de datos. Arrastrá el archivo o seleccionalo.
